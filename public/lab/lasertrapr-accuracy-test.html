lasertrapr accuracy test

I tested our single molecule analysis against the SPASM (Greenberg 2020) simulated data sets. These data have a ‘key’ that identified the exact data point where events start and stop. Using our program, I estimated the start event start/stop through the combination of event identification through HM-Model and changepoint. By subtracting real transition points given by the simulated data set key from the estimated start or end of the event given by the program we cam see how accurate our program is.

A positive value corresponds to our program being late and missing the start and if the program is early that will correspond to a negative value (i.e. program says an event starts at data point #1005 where the key has it starting at #1000, 1005 - 1000 = +5)
